# iFlagTrivia: A Flag Quiz App

iFlagTrivia is a fun and engaging trivia quiz app built using **UIKit** where players are shown a flag and asked to identify which country's flag it is. Players select their answer from four options, and the app provides feedback on whether the answer is correct or incorrect. This app uses **emoji flags** available in the Apple emoji set to make it easy to integrate a variety of flags.

## Features

- **Quiz Mode**: Players are presented with a random flag and four country options to choose from.
- **Emoji Flags**: The app uses emoji flags that are part of the Apple emoji set for simplicity and easy flag integration.
- **Score Tracking**: Keep track of the score as the user progresses through the quiz.
- **Programmatic UI**: The app is entirely built using UIKit programmatically, without the use of storyboards.

## Technologies Used

- **UIKit** (Programmatic UI)
- **Swift**
- **Apple Emoji Set** (for flag images)
- **Xcode 12+**

## Installation

To run the app, clone the repository and open the project in Xcode:

```bash
git clone https://github.com/your-username/iTrivia.git
cd iTrivia
open iFlagTrivia.xcodeproj
```
